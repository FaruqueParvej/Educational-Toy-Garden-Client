## Getting Started

### Prerequisites

Install the dependencies:

- npm
  ```sh
  npm install
  ```

- Clone the repo
  ```sh
  git clone https://github.com/programming-hero-web-course-4/b7a11-toy-marketplace-client-side-FaruqueParvej
  ```
- Install NPM packages
  ```sh
  npm install
  ```
